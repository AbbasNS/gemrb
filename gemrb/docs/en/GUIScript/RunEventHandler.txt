
Prototype: GemRB.RunEventHandler(String[, error])

Description:  Executes a GUIScript event handler function named String. If error is set to nonzero, then a missing handler will cause an error.

Parameters: 
String - GUIScript event handler function
error - (optional) if nonzero, a missing handler will cause an error

Return value: N/A

See also: ExecuteString
