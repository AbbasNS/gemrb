
Prototype: GemRB.SetTAAutoScroll(WindowIndex, ControlIndex, State)

Description: Sets the TextArea auto-scroll feature status. Some TextAreas are autoscrolling while text was written into them, thus always showing the last content.

Parameters: WindowIndex, ControlIndex  - the window/control index of the textarea
State - boolean, whether the TextArea should have the autoscroll feature or not.

Return value: N/A

See also: MoveTAText, SetTextAreaSelectable, TextAreaClear, TextAreaAppend, accessing_gui_controls
